## Summary
# What is Agentic AI?

## Direct Answer  
Agentic AI refers to AI systems—often built around [[Large Language Models]] (LLMs)—that autonomously plan, decide, and act toward complex goals. These systems operate in iterative loops by observing outputs, adapting strategies, and utilizing tools and memory to achieve tasks beyond single-shot LLM responses.

---

## Key Facts

- **Core Concept:**  
  An *agent* in agentic AI = LLM + tools + memory + goal-driven action loop.

- **Functionality:**  
  - Autonomously perform multi-step reasoning and planning.  
  - Use external tools such as web search, code execution, API calls, and file operations.  
  - Employ different memory types such as in-context prompts, external databases/files, and episodic memory (conversation history).

- **Architecture:**  
  Often involves orchestrating multiple agents (multi-agent systems) where an orchestrator coordinates specialized sub-agents.

- **Operational Patterns:**  
  - ReAct (Reason + Act): iterative reasoning and action cycles.  
  - Plan-and-Execute: decomposing goals into executable plans.  
  - Reflexion: self-review and error correction loops.  
  - Tool-use loops: invoking external APIs or systems as part of decision-making.

- **Challenges:**  
  - Managing hallucinations and inaccurate outputs.  
  - Avoiding infinite loops in decision/action cycles.  
  - Handling errors from tool integration and context window limitations.

- **Examples:**  
  - AutoGPT, Claude Code, Devin, and other custom orchestration pipelines implementing agentic behavior.

---

## Relevant Subtopics

- **Large Language Models (LLMs):**  
  Agentic AI harnesses the language understanding and generation capabilities of LLMs as the “brain” of autonomous agents.

- **Artificial Intelligence:**  
  Agentic AI is a subfield focusing on autonomy within AI, moving beyond fixed-task models to systems that adapt and self-direct.

- **Prompt Engineering:**  
  Crafting precise prompts is crucial in agentic AI to guide the agent’s decision-making, tool usage, and reasoning processes effectively.

- **Transformer Models:**  
  The backbone architecture powering LLMs that enables [[Agentic Systems]] to process and generate complex language-based instructions.

---

*In summary*, agentic AI builds on large language models by integrating tools, memory, and goal-oriented loops, enabling autonomous, adaptive, and multi-step problem solving across diverse, complex tasks.
