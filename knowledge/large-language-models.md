# Large Language Models

**Created**: 2026-04-13  
**Updated**: 2026-04-13  
**Tags**: llm, transformers, nlp, deep learning, gpt

## Summary
Large Language Models (LLMs) are deep learning models trained on vast amounts of text data to understand and generate human language. They are built on the transformer architecture and use self-attention mechanisms to model relationships between tokens across long contexts.

## Key Points
- LLMs are trained via next-token prediction on massive text corpora (books, web, code)
- The transformer architecture (2017) is the foundation — uses attention instead of recurrence
- Scale is key: more parameters + more data + more compute = better capabilities
- Prominent LLMs include GPT-4, Claude, Gemini, Llama, and Mistral
- Fine-tuning and RLHF (Reinforcement Learning from Human Feedback) align models to human preferences
- LLMs can reason, write code, summarize, translate, and answer questions
- Context window determines how much text the model can process at once
